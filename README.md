# Student-task-Manager
A C++ console task manager with SQLite database

## Features
- Adds tasks with title, due date and priority
- Views all tasks
- Marks tasks as Done
- Delete tasks
- Filters tasks based on priority
- Views pending tasks
- Searches task with keywords

## Technologies Used
- C++
- SQLite3
- Git

## How to run

### Requirements
- g++ compiler
- SQLite3 amalgamation files

### Steps
1. Clone the repository
2. Compile: `g++ src/main.cpp sqlite3.o -o taskmanager`
3. Run: `./taskmanager.exe`
